# TestProject
Как запустить проект:

1. Клонируйте репозиторий или скачайте файлы.

2. Откройте index.html в любом браузере (Chrome, Firefox, Edge и т.д.).

3. Приложение сразу загрузит список задач с внешнего API (https://jsonplaceholder.typicode.com/todos) и пользователей (https://jsonplaceholder.typicode.com/users).

Технологии:
HTML5
CSS3
JavaScript

Что можно доработать при наличии большего времени:

1. Реализовать добавление, редактирование и удаление задач.

2. Сделать полностью адаптивный дизайн и улучшить визуальные эффекты.

3. Добавить debounce для поиска.

4. Добавить кнопку сброса фильтров.

5. Начать использовать Media Queries.
